# College-ERP

🚀 A smarter, modern ERP system for our college.

## Tech Stack
- Frontend: React.js + Tailwind CSS
- Backend: Flask
- Database: PostgreSQL

## Timeline
We aim to build this by **31st December 2025**, with active development starting in **August**.

## Modules (Planned)
- 🧑‍🎓 Student Login
- 📊 Dashboard
- 📅 Timetable & Attendance
- 📝 Feedback & Requests
